# Antigravity Academy MVP

¡Bienvenido al futuro de la educación galáctica! Esta plataforma está diseñada para enseñarte a construir sistemas de agentes de IA utilizando la arquitectura de 4 capas de Antigravity.

## Arquitectura del Proyecto

El proyecto sigue una estructura de **Single Page Application (SPA)** moderna construida con:
- **Vue 3 (Composition API):** Lógica reactiva modular.
- **Tailwind CSS:** Diseño premium con estética "Deep Space".
- **Pinia:** Gestión de estado persistente para XP, niveles y progreso.
- **Vue Router:** Navegación fluida entre niveles de misión.

## Instrucciones de Instalación

1. Navega a la carpeta del proyecto:
   ```bash
   cd .tmp/outputs/antigravity-academy
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

2. **Orquestador:** Cómo ordenar el flujo de trabajo.
3. **Agentes:** Cómo asignar herramientas especializadas.
